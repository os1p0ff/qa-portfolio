## Создание симулятора в Xcode

### Цель

Создать виртуальное устройство iPhone для тестирования приложений без физического устройства.

### Шаг 1. Установить Xcode

1. Открыть App Store на macOS.
2. Найти приложение Xcode.
3. Установить Xcode и дождаться завершения установки.

Важно:
- Xcode занимает более 15 ГБ дискового пространства.
- Рекомендуется использовать актуальную версию Xcode.

### Шаг 2. Запустить Xcode

1. Открыть Xcode.
2. Дождаться завершения первоначальной настройки.
3. При необходимости установить дополнительные компоненты.

Важно:
- Не закрывать Xcode во время установки компонентов.
- Проверить наличие актуальных SDK.

### Шаг 3. Открыть менеджер симуляторов

Перейти:

```text
Window → Devices and Simulators
```

или

```text
Xcode → Open Developer Tool → Simulator
```

### Шаг 4. Создать новый симулятор

1. Перейти на вкладку Simulators.
2. Нажать кнопку "+".
3. Выбрать модель устройства (например, iPhone 16).
4. Выбрать версию iOS.
5. Нажать Create.

Важно:
- Использовать актуальную версию iOS.
- Убедиться, что нужный Runtime установлен.

### Шаг 5. Запустить симулятор

1. Выбрать созданное устройство.
2. Нажать Start или открыть приложение Simulator.
3. Дождаться загрузки iOS.

Важно:
- Проверить корректный запуск системы.
- Убедиться, что симулятор отображается в списке устройств Xcode.

### Возможные проблемы

| Проблема | Решение |
|-----------|----------|
| Нет нужной версии iOS | Скачать Runtime через Settings → Platforms |
| Симулятор не запускается | Перезапустить Xcode |
| Недостаточно места на диске | Освободить место и удалить ненужные Runtime |
| Медленная работа | Закрыть лишние приложения и перезапустить симулятор |

### Вывод

Симулятор Xcode позволяет тестировать iOS-приложения без физического устройства. Для успешной работы важно использовать актуальную версию Xcode, установить необходимые Runtime и проверять совместимость версии iOS с тестируемым приложением.
