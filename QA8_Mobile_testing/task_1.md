## 1. Анализ сервисов распространения тестовых сборок iOS

### Цель

Изучить способы распространения тестовых версий приложения среди команды разработки и тестирования и выбрать наиболее подходящий вариант для стартапа с ограниченным бюджетом.

### Сравнительная таблица

| Сервис | Стоимость | Простота настройки для разработчика | Простота установки для тестировщика | Ограничения |
|---|---|---|---|---|
| **TestFlight** | Apple Developer Program — 99 USD/год | Средняя — настройка через App Store Connect и сертификаты | Очень высокая — установка через приложение TestFlight | Проверка внешних 
сборок, срок действия билдов |
| **Firebase App Distribution** | Бесплатно | Высокая — интеграция через Firebase Console, CLI или CI/CD | Высокая — установка по приглашению или ссылке | Требуется инфраструктура подписи Apple |
| **Diawi** | Бесплатный тариф + платные планы | Очень высокая — загрузка IPA и генерация ссылки | Средняя — настройка доверия сертификату | Ограничения бесплатного тарифа и управления доступом |

---

### Рекомендация по выбору сервиса

Для стартапа рекомендуется использовать **Firebase App Distribution**.

#### Причины выбора

- Минимальные расходы.
- Быстрая доставка сборок.
- Интеграция с CI/CD.
- Удобная работа для команды QA.
- Возможность масштабирования проекта.

#### Когда выбрать TestFlight

Использовать при подготовке релиза и внешнем тестировании.

### Сценарии тестирования

#### Сценарий 1. Потеря сети во время перевода

**Предусловия:**  
Пользователь авторизован.

**Шаги:**

1. Открыть экран перевода.
2. Ввести сумму.

## 3. Сообщение коллеге об утечках памяти

Привет.

Изучил тему утечек памяти применительно к нашему финансовому приложению.

### Как проявляется утечка памяти

#### Запуск приложения

Увеличивается время запуска.

#### Работа на главном экране

Появляются подвисания интерфейса.

#### Переход между экранами

Память растёт после каждого перехода.

#### Фоновый режим

Приложение чаще выгружается системой.

#### Возврат из background

Увеличивается время восстановления.

#### Выполнение операций

В памяти остаются временные данные транзакций.

#### Длительная работа

Снижается производительность.

#### Работа при нехватке памяти

iOS может завершить приложение.

### Инструменты диагностики

- Instruments → Leaks
- Instruments → Allocations
- Xcode Memory Graph
- Профилирование длительных сценариев

## 4. Что такое утечка памяти?

**Утечка памяти** — это ситуация, при которой приложение продолжает удерживать память после того, как объекты перестали использоваться.

### Проявления

- замедление приложения;
- рост потребления памяти;
- зависания;
- увеличение расхода батареи;
- аварийное завершение.

### Частые причины в iOS

- retain cycle;
- сильные ссылки;
- неосвобождённые closures;
- неочищенный кэш;
- фоновые процессы.

Для финансового приложения утечки памяти особенно опасны, так как влияют на стабильность денежных операций и пользовательский опыт.
