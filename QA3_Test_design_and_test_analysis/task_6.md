# Задание 6. Матрица тестового покрытия.

Страница проверки: https://21-school.ru/methodology  
Раздел: «Как мы учим»

| Функциональность | Категория | Обоснование | Номер проверки |
|---|---|---|---|
| Отображение страницы «Как мы учим» | Критический | Если страница не загружается, пользователь не сможет узнать информацию об обучении | TC-01, TC-02 |
| Основной текст раздела | Критический | Основная цель страницы — объяснить пользователю процесс обучения | TC-03, TC-04 |
| Описание этапов обучения | Критический | Пользователь должен понимать структуру обучения | TC-05, TC-06 |
| Внутренние ссылки страницы | Важный | Пользователь может переходить к дополнительной информации | TC-07, TC-08 |
| Навигация между разделами сайта | Важный | Удобная навигация помогает пользователю находить нужную информацию | TC-09 |
| Отображение изображений и иллюстраций | Важный | Визуальные элементы помогают лучше понять информацию | TC-10 |
| Кроссплатформенность | Важный | Страница должна корректно работать в разных браузерах | TC-11, TC-12 |
| Адаптивность страницы | Важный | Страница должна корректно отображаться на мобильных устройствах | TC-13, TC-14 |
| Скорость загрузки страницы | Вспомогательный | Медленная загрузка может ухудшить пользовательский опыт | TC-15 |
| Корректность отображения шрифтов и оформления | Вспомогательный | Влияет на внешний вид страницы, но не на основную функциональность | TC-16 |
