---

## 🧠 Мои навыки

### 🔹 Ручное тестирование
- Составление тест-планов, чек-листов, тест-кейсов
- Составление баг-репортов (Jira / YouTrack)
- Техники тест-дизайна:
  - классы эквивалентности
  - граничные значения
  - попарное тестирование (pairwise)

### 🔹 Веб-тестирование
- Chrome DevTools (Network, Coverage, Security)
- Валидация HTML через W3C Validator
- HTTPS, SSL/TLS-сертификаты
- UI-тестирование

### 🔹 Бэкенд-тестирование
- REST API (Postman, автотесты, коллекции)
- HTTP-статусы
- Базы данных: SQL / NoSQL

### 🔹 Мобильное тестирование
- Эмуляторы: Android Studio, Xcode Simulator
- Charles Proxy (перехват, подмена, замедление трафика)
- Логирование (Logcat)

### 🔹 Безопасность
- XSS, SQL-инъекции
- Криптография: хеширование, шифрование
- Инструменты: Burp Suite, OWASP ZAP

### 🔹 Интеграция с AI и чат-боты
- Проектирование AI-агентов в n8n
- Интеграция с Telegram API
- Создание карьерного консультанта на базе LLM
- Парсинг данных (RSS, HH.ru) и генерация дайджестов через AI
