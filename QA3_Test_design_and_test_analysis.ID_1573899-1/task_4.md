## 1. Количество всех возможных комбинаций (Полный перебор)

Чтобы узнать общее количество комбинаций без применения техник тест-дизайна, необходимо перемножить количество вариантов для каждого параметра:

- **Возраст:** 3 значения (18–35 / 36–55 / 56–75)
- **Наличие задолженностей:** 2 значения (да / нет)
- **Тип трудовых отношений:** 3 значения (ИП / Самозанятый / Бессрочный договор)
- **Наличие действующих кредитов:** 2 значения (да / нет)

**Формула:** `3 * 2 * 3 * 2 = 36`

**Вывод:** Методом «каждое с каждым» мы получили бы **36 тест-кейсов**. Это избыточно для данной функциональности, поэтому применим технику попарного тестирования.
