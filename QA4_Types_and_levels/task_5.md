# Задание 5. Grey-box тестирование

У тестировщика есть доступ к базе данных магазинов.

## Чек-лист проверок

### Проверка данных магазинов
- Проверить что магазины из базы данных отображаются на карте.
- Проверить соответствие координат магазина в базе и на карте.
- Проверить корректность названия магазина.

### Проверка поиска
- Проверить что поиск по названию соответствует данным из базы.
- Проверить поиск по частичному совпадению названия.
- Проверить поиск по типу магазина.

### Проверка результатов поиска
- Проверить что найденные магазины присутствуют в базе.
- Проверить что отсутствующие в базе магазины не отображаются.

### Проверка маршрута
- Проверить что маршрут строится до координат магазина из базы данных.
- Проверить что маршрут строится до ближайшего магазина выбранного типа.

### Проверка сохранения данных
- Проверить что предыдущие поиски сохраняются.
- Проверить что данные сохраняются корректно в базе.
