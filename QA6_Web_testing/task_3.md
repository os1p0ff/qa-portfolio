# Task 3

## Кроссбраузерное тестирование

Было проведено тестирование сайта в следующих браузерах:

- Google Chrome
- Yandex Browser
- Microsoft Edge

В процессе проверки сравнивались:

- отображение header и footer
- расположение блоков на странице
- отображение изображений
- работа кнопок и ссылок
- адаптивность интерфейса
- корректность шрифтов и отступов

По результатам тестирования визуальных и функциональных отличий между браузерами обнаружено не было.

Во всех браузерах:
- верстка отображается корректно
- элементы интерфейса работают одинаково
- смещения блоков и ошибок отображения не обнаружено
- изображения и стили загружаются корректно

## Вывод

Сайт успешно прошел базовое кроссбраузерное тестирование.
