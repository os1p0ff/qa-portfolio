## 1. Изменение названия раздела в хедере

**Действие:**  
Открыт сайт https://21-school.ru/ в Google Chrome. С помощью DevTools (вкладка Elements) изменён текст одного из разделов хедера.  

**Пример:**  
Вместо «Бесплатная школа цифровых технологий» в хедере указано «ЗАДАНИЕЕЕЕЕЕЕЕЕЕЕ».

**Результат:**  
Изменение отобразилось на странице мгновенно (локально, без сохранения на сервере).

## 2. Неиспользуемые CSS и JS

**Инструмент:**  
Вкладка Coverage (More tools → Coverage).

**Действие:**  
Запущена запись покрытия кода, страница перезагружена. Проанализированы загруженные CSS и JS файлы.

**Результаты:**  
- Значительная часть CSS и JS не используется при первом открытии страницы (красные/серые полосы в Sources).  
- Это увеличивает время загрузки и потребление трафика.

**Скриншот:**  
`task_2.2-2.11`

## 3. Замеры скорости загрузки

**Условия:**
- Для каждого замера включена опция «Disable cache»
- Использована вкладка Network
- Измерялись `DOMContentLoaded` и `Load`

| Скорость | DOMContentLoaded | Load | Передано данных | Ресурсов | Количество запросов |
|----------|-----------------|------|----------------|----------|--------------------|
| **Fast 4G** | 7.23 с | 57.29 с | 9.2 MB | 13.6 MB | 134 |
| **Slow 4G** | 8.41 с | 41.39 с | 7.6 MB | 13.6 MB | 134 |
| **300 kbit/s** | 26.54 с | 408 с (6.8 мин) | ~7.6 MB | ~13.6 MB | ~134 |

### Детали замеров

#### Fast 4G
- DOMContentLoaded: 7.23 с
- Load: 57.29 с
- Передано: 9.2 MB
- Ресурсов: 13.6 MB
- Запросов: 134
- Finish: 58.23 с

#### Slow 4G
- DOMContentLoaded: 7.23 с
- Load: 41.39 с
- Передано: 7.6 MB
- Ресурсов: 13.6 MB
- Запросов: 134
- Finish: 41.39 с

#### 300 kbit/s (эмуляция)
- DOMContentLoaded: ~26.54 с
- Load: ~408 с (6 минут 48 секунд)
- Передано: ~7.6 MB
- Ресурсов: ~13.6 MB
