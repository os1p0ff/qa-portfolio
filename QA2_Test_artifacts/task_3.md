## 1. Цель тестирования

Проверить корректность работы раздела «Корпоративное обучение», его отображение и соответствие заявленной информации.

## 2. Объект тестирования

Веб-страница раздела «Корпоративное обучение» на сайте «Школа 21».

## 3. Область тестирования

В рамках тестирования проверяется:

- Открытие страницы без ошибок  
- Корректность отображения текста  
- Работа кнопок и ссылок  
- Работа формы (если присутствует)  
- Отображение изображений  
- Адаптивность на мобильном устройстве  

## 4. Типы тестирования

- Функциональное тестирование  
- UI тестирование  
- Проверка адаптивности  
- Регрессионное тестирование (повторное тестирование) 

## 5. Тестовая среда

- ОС: Windows 10  
- Браузер: Google Chrome  
- Мобильное устройство: Android (Chrome)  

## 6. Критерии начала тестирования

- Страница доступна по ссылке  
- Сайт корректно загружается  


## 7. Критерии завершения тестирования

- Все проверки выполнены  
- Критические ошибки отсутствуют  
- Составлен баг-репорт при наличии дефектов  

## 8. Результат тестирования

По итогам тестирования будет сделан вывод о готовности раздела к использованию.
