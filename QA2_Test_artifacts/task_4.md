# Задание 4. Чек-лист раздела «Как мы учим» (Сайт Школы 21)

## Чек-лист проверки контента и функциональности

### Тестовые данные
- Site: https://21-school.ru/methodology

## Навигация

| Проверка | Статус |
|----------|--------|
| Страница "Как мы учим" открывается без ошибок | pass |
| Переход на страницу происходит через главное меню в разделе "Поступающим" | pass |
| При обновлении страницы данные не теряются, пользователь находится на том же разделе | pass |

## Контент

| Проверка | Статус |
|----------|--------|
| Заголовок страницы соответствует разделу "Как мы учим" | pass |
| Текст отображается полностью, без обрезаний и наложений | pass |
| Отсутствуют орфографические и пунктуационные ошибки | pass |
| Информация соответствует методологии обучения "Школы 21" | pass |
| Информация об организации и формате обучения представлена понятно (NEW) | pass |
| Описание методики обучения прозрачное и вызывает доверие (NEW) | pass |
| Контент структурирован: есть подзаголовки, логические блоки, визуальные акценты (NEW) | pass |
| Информация актуальна (указана дата обновления или подтверждена актуальность) (NEW) | fail |

## Авторизация

| Проверка | Статус |
|----------|--------|
| При нажатии на кнопку "Вход" перебрасывает пользователя на страницу авторизации | pass |
| При нажатии на кнопку "Регистрация" перебрасывает на форму подачи заявки | pass |
| При нажатии на кнопку "Войти по почте" перебрасывает на страницу авторизации | pass |
| При нажатии на кнопку "Войти по Сбер ID" перебрасывает на страницу авторизации Сбер ID | pass |
| Все способы авторизации работают корректно | pass |

## Функциональность 

| Проверка | Статус |
|----------|--------|

| Внутренние ссылки ведут на корректные страницы | pass |
| Все ссылки открываются корректно | pass |

## Визуальная часть

| Проверка | Статус |
|----------|--------|

| При изменении масштаба ничего не ломается, информация корректно адаптируется | pass |
| На мобильном устройстве страница отображается корректно и оптимизирована | pass |

## Доверие и социальные доказательства (NEW раздел)

| Проверка | Статус |
|----------|--------|
| Есть возможность перейти к контактной информации | pass |
| На странице присутствуют отзывы студентов или истории выпускников | fail |
| Есть примеры проектов или достижений студентов | fail |
| Страница формирует доверие к образовательной организации | pass |

## Примечания

- **fail** на пункте актуальности информации: на странице нет явной даты обновления.  
- **fail** на социальных доказательствах: отзывы и кейсы студентов есть на других страницах, но не на текущей.
