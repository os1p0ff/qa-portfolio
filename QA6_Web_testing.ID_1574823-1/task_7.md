## Результаты валидации

### Общее количество ошибок
~15 ошибок (повторяющихся типов)

### Количество критичных ошибок
**0** — критичных ошибок, полностью ломающих отображение сайта, не обнаружено.

### Самые частые типы ошибок

| Тип ошибки | Количество вхождений | Описание |
|------------|---------------------|----------|
| `Element p not allowed as child of element span` | 4 | Тег `<p>` вложен внутрь `<span>`, что запрещено спецификацией |
| `Element p not allowed as child of element h2` | 8 | Тег `<p>` вложен внутрь `<h2>` |
| `Element ul not allowed as child of element ul` | 2 | Вложенный `<ul>` без `<li>` |
| `Heading h4 follows h2, skipping 1 level` | 1 | Нарушение иерархии заголовков (пропущен h3) |

### Наличие ошибок доступности (Accessibility)

| Ошибка | Влияние на доступность |
|--------|----------------------|
| Нарушение иерархии заголовков (h2 → h4) | Скринридеры могут запутаться в структуре страницы |
| Вложенные `<ul>` без `<li>` | Нарушает навигацию для пользователей с вспомогательными технологиями |
| `<p>` внутри `<span>` | Может вызвать некорректное озвучивание структуры контента |

**Вывод:** Есть проблемы доступности средней серьезности.

## Приоритизация ошибок

### Высокий приоритет (исправить срочно)
- **Нет** — критичных ошибок, ломающих сайт, не обнаружено

### Средний приоритет (исправить в ближайшее время)
- Нарушение иерархии заголовков (h2 → h4) — влияет на SEO и доступность
- Вложенные `<ul>` без `<li>` — нарушает семантику списков
- `<p>` внутри `<span>` и внутри `<h2>` — некорректная структура

### Низкий приоритет (можно отложить)
- Информационные сообщения о trailing slash (`<img src="..." />`) — не влияют на работу
- Предупреждение о CSP и `report-uri` — техническое
- Атрибут `type="text/javascript"` на скриптах — избыточен в HTML5
