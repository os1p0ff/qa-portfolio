## 1. Анализ cookies

Анализируемый сайт: https://www.ozon.ru/

| Имя cookie | Тип | Предполагаемое назначение |
|------------|------|---------------------------|
| __Secure-access-token | Authentication | Хранение токена авторизации пользователя |
| __Secure-refresh-token | Authentication | Обновление пользовательской сессии |
| __Secure-user-id | Functional | Хранение идентификатора пользователя |
| ob_theme | Preferences | Сохранение выбранной темы интерфейса |
| __Secure-ab-group | Analytics | Участие пользователя в A/B тестировании |

## 2. Анализ флагов безопасности cookies

Были исследованы cookies, связанные с авторизацией пользователя.

| Cookie | Secure | HttpOnly | SameSite | Оценка безопасности |
|--------|---------|-----------|-----------|---------------------|
| __Secure-access-token | Да | Да | Lax | Настройка безопасна. Cookie передается только по HTTPS и недоступен JavaScript, что снижает риск XSS-атак |
| __Secure-refresh-token | Да | Да | Lax | Безопасная конфигурация для хранения refresh-токена |
| __Secure-user-id | Да | Да | Lax | Достаточно безопасная настройка для идентификатора пользователя |
